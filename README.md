# laravue-admin

> 微型社区 LaraVue 的后台管理界面

## 功能

> 根据权限动态生成前端路由

> 页面级和按钮级权限的精确控制。

> 页面级权限由 vue 动态加载路由实现；按钮级权限没有的话，按钮显示灰色且为不可点击状态。
